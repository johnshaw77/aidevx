# 🤫 假裝工作中 - 終極摸魚神器

一個讓你看起來超級忙碌的網站工具！完美模擬各種工作場景，讓你在摸魚時也能保持專業形象。

## ✨ 功能特色

### 🎭 **四種工作模式**
1. **📊 Excel 數據分析模式**
   - 逼真的 Excel 界面和數據表格
   - 動態計算進度顯示
   - 自動生成的銷售數據

2. **💻 程式碼編輯器模式**
   - Visual Studio Code 風格界面
   - 自動打字的 Python 代碼
   - 真實的文件樹結構

3. **📧 郵件處理模式**
   - Outlook 風格郵件界面
   - 自動撰寫專業郵件內容
   - 收件匣和文件夾結構

4. **📝 會議記錄模式**
   - Teams 會議記錄界面
   - 自動生成會議內容
   - 專業的會議格式

### ⌨️ **快捷鍵操作**
- **Space** - 切換工作模式
- **B** - 老闆鍵（緊急切換到正經文件）
- **H** - 隱藏/顯示快捷鍵提示

### 🎯 **智能特效**
- **自動打字動畫** - 模擬真實打字過程
- **隨機工作事件** - 模擬通知、進度更新
- **忙碌指示器** - 持續顯示工作狀態
- **鼠標活動檢測** - 移動鼠標觸發工作模擬

## 🎮 使用方式

### 🚀 **快速開始**
1. 打開網站
2. 按 `Space` 鍵切換不同工作模式
3. 移動鼠標模擬工作活動
4. 緊急時刻按 `B` 鍵使用老闆鍵！

### 🛡️ **老闆鍵功能**
當老闆突然出現時，立刻按 `B` 鍵：
- 瞬間切換到正經的公司政策文件
- 頁面標題變成「重要政策文件」
- 內容包含營運目標、人力資源規劃等
