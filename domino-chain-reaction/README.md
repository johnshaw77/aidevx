# 🎯 3D 骨牌連鎖反應遊戲

一款沉浸式 3D 互動遊戲，讓你自由放置骨牌，然後點擊啟動觀看逼真的物理連鎖反應！

## ✨ 核心功能

### 🖱️ 直覺的放置系統
- **左鍵點擊** — 在地面上放置彩色骨牌
- 每個骨牌隨機著色（紅色或綠色）
- 支援自由度的放置位置和角度

### ⚙️ 完整的物理模擬
- **Cannon.js** 物理引擎
- 逼真的重力、碰撞、摩擦力
- 支援動態調整重力強度（0~50）

### 🎮 靈活的參數控制
- **重力調整** — 範圍 0~50，改變連鎖反應劇烈程度
- **骨牌高度** — 3~8 單位
- **骨牌厚度** — 0.2~1.0 單位

### 🎥 3D 視角控制
- **右鍵拖曳** — 自由旋轉視角
- **滑輪縮放** — 放大/縮小場景
- 支援 360° 全方位觀看

### 📊 實時統計
- 骨牌總數
- 已倒下骨牌數
- 實時 FPS 監控

## 🚀 操作說明

1. **放置骨牌**
   - 點擊「放置骨牌」啟用模式
   - 左鍵點擊地面放置骨牌
   - 建議放置 10~50 個骨牌形成連鎖路徑

2. **調整參數**
   - 使用滑塊調整重力、高度、厚度
   - 參數實時生效

3. **啟動連鎖反應**
   - 點擊「啟動連鎖反應」
   - 系統隨機選擇一個骨牌施加強推力
   - 觀看爽快的連鎖倒塌

4. **視角控制**
   - 右鍵拖曳改變視角
   - 滑輪縮放距離
   - 找到最佳觀看角度
