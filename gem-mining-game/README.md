# 💎 寶石礦山 - Gem Mining Game

一個帶有**粒子效果與升級系統**的互動式點擊挖礦遊戲！

## 🎮 核心機制

🪨 **點擊挖礦**
- 每次點擊獲得金幣
- 隨機掉落 4 種寶石（紅、藍、綠、紫）
- 粒子特效與浮動數字反饋

🔗 **連擊系統**
- 連擊 3 次觸發連擊加成
- 加成倍數最高 2.5 倍
- 5 秒內未點擊重置

⚡ **升級功能**
- **強化鎬**：每次 +1 傷害 (10 金幣)
- **連擊加成**：增加倍數上限 (50 金幣)
- 動態升級按鈕狀態

## ✨ 視覺設計

- **深藍科幻背景** + 星空效果
- **Canvas 粒子系統**：金幣、寶石浮動動畫
- **浮動數字**：即時傷害反饋
- **Neon 邊框** + 陰影發光效果
- **響應式佈局**：桌面雙欄 / 手機單欄

## 🚀 玩法

1. **點擊大圓形鎬**獲得金幣
2. 達到 **連擊 3 次**時觸發加成
3. **快速連擊**維持連擊倍數，最高 2.5 倍！
4. 用金幣**升級鎬**或**連擊加成**
5. 收集 4 種寶石，記錄你的礦山成就
