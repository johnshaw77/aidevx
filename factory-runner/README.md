# Factory Runner — 智慧工廠生產線遊戲

一款以智慧製造為主題的策略管理小遊戲，玩家需要平衡產量與品質，獲得最高分數。

## 🎮 遊戲玩法

### 目標
在 60 秒內管理 FPC 生產線，透過控制傳送帶速度、品質檢測和設備維修，獲得最高分數。

### 控制項目
1. **傳送帶速度** — 影響產量和不良率
   - 慢速：產量低但品質穩定
   - 中速：平衡產量與品質
   - 快速：高產量但不良率增加

2. **品質檢測** — 降低不良率但影響產量
   - 開啟：不良率降低 70%，但產量受影響
   - 關閉：最大產量但不良率提升

3. **設備維修** — 維持機器健康
   - 機器健康度影響產品品質
   - 及時維修避免大量不良品

### 計分機制
```
分數 = 產量 × 良率 × 速度加成 × 品檢加成
```

- **產量**：生產的總件數
- **良率**：(總件數 - 不良品) / 總件數
- **速度加成**：慢速 0.5x / 中速 1x / 快速 1.5x
- **品檢加成**：開啟品檢 1.2x

## 🏭 遊戲特色

- **工廠主題**：結合 FPC 製造背景
- **策略性**：需要平衡多個變數
- **視覺效果**：動畫傳送帶和產品流動
- **即時反饋**：指示燈顯示設備狀態
- **學習性**：模擬真實生產管理場景

## 💡 策略提示

1. **初期建議**：使用中速，開啟品檢建立穩定基礎
2. **中期策略**：根據機器狀態調整速度
3. **後期衝刺**：適當承受風險提升速度
4. **維修時機**：機器健康度低於 50% 時立即維修

## 🔧 技術實現

- **前端**：純 HTML5 + CSS3 + JavaScript
- **動畫**：CSS 關鍵影格和 JavaScript 計時器
- **響應式**：適配不同螢幕尺寸
- **無依賴**：不需要額外框架或庫
