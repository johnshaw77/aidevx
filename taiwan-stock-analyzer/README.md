### 📊 **指標解讀**

#### **RSI 相對強弱指標**
- **70以上**：股票可能超買，注意回檔風險
- **30以下**：股票可能超賣，留意反彈機會
- **50附近**：處於中性區間

#### **MACD 指標**
- **正值且上升**：多頭趨勢，持續看多
- **負值且下降**：空頭趨勢，保持觀望
- **黃金交叉**：短期均線上穿長期均線，買進訊號

#### **KD 指標**
- **K>D且向上**：多方力道強勁
- **K<D且向下**：空方力道占優
- **高檔鈍化**：持續強勢，注意轉折

#### **威廉指標**
- **-20以上**：超買區域，注意賣出時機
- **-80以下**：超賣區域，留意買進機會
- **-20到-80**：正常波動區間
