# 🕰️ 時光膠囊聊天室

## 專案描述

**時光膠囊聊天室** 是一個讓用戶可以寫訊息給未來的自己或他人的互動式網頁應用。

### ✨ 核心功能

- **📝 創建時光膠囊**：寫下想對未來說的話
- **⏰ 靈活時間設定**：支援分鐘、小時、天、月或自定義時間
- **🔒 時間鎖定機制**：膠囊在指定時間前無法開啟
- **✨ 開啟特效**：首次開啟時的時光旅行動畫效果
- **💾 本地存儲**：使用瀏覽器 localStorage 持久化數據

### 🎨 設計特色

- **星空背景**：動態星星閃爍效果
- **漸層設計**：現代化的漸層色彩搭配
- **響應式佈局**：完美適配各種設備
- **動畫效果**：豐富的 CSS 動畫和過渡效果
- **直觀操作**：簡潔明瞭的使用者介面

### 🚀 使用方式

1. **創建膠囊**
   - 填寫收件人（如：未來的自己）
   - 寫下想說的話
   - 設定開啟時間
   - 點擊「封存到時光膠囊」

2. **查看膠囊**
   - 切換到「我的膠囊」分頁
   - 查看所有已創建的膠囊
   - 等待時間到達後開啟膠囊

3. **膠囊狀態**
   - 🔒 鎖定中：顯示倒計時
   - ✨ 可開啟：時間已到
   - 📖 已開啟：已經查看過的膠囊

### 💡 使用場景

- **自我激勵**：給未來的自己寫鼓勵信
- **目標設定**：記錄當下的願望和計劃
- **情感寄託**：在特殊時刻留下回憶
- **朋友互動**：給朋友的未來驚喜
- **時間見證**：記錄重要時刻的想法

### 🛠️ 技術特點

- **純前端實現**：HTML + CSS + JavaScript
- **無需後端**：完全基於瀏覽器本地存儲
- **現代化 CSS**：使用 Grid、Flexbox 和動畫
- **用戶友好**：流暢的交互體驗
