# FPC 良率預測系統 🔬

基於人工智慧的軟性印刷電路板（FPC）良率預測與最佳化系統。

## 🌟 專案特色

- **AI 良率預測**：基於製程參數即時預測生產良率
- **參數最佳化**：智慧分析並提供製程改善建議
- **即時監控**：模擬多產線即時狀態監控

- **溫度控制**：40-60°C（最佳：50°C）

- **溫度控制**：60-80°C（最佳：70°C）
- **處理時間**：5-15分鐘（最佳：10分鐘）

### 環境條件
- **濕度控制**：40-60%（最佳：50%）
- **振動監控**：1-10級（最佳：3級）

## 🎛️ 操作說明

### 1. 參數調整
拖動滑桿調整各項製程參數，系統將即時計算良率預測。

### 2. 預測分析
觀察良率圓環圖與信心度條，了解當前設定的預期效果。

### 3. 最佳化建議
查看系統生成的智慧建議，優化製程參數設定。

### 4. 趨勢分析
檢視歷史良率趨勢，掌握生產品質變化。

### 5. 即時監控
監控多條產線即時狀態，及時發現異常。
