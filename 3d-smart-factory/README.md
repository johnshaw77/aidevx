# 🏭 3D 智慧工廠視覺化

一個沉浸式的智慧工廠 3D 視覺化系統，專為 FPC（軟性印刷電路板）生產線設計。結合了即時數據監控、互動式 3D 場景和工業 4.0 概念。

## ✨ 核心特色

### 🎯 真實工廠元素
- **FPC 生產線**：6 個工作站模擬真實軟板製造流程
- **機械手臂**：3 台自動化機器人執行精密作業
- **AGV 小車**：2 台自主導航車輛負責物料運輸
- **智慧照明**：狀態指示燈即時反映設備運行狀況

### 📊 即時數據監控
- **產線效率**：動態顯示生產效率百分比
- **設備狀態**：即時監控所有設備運行狀況
- **生產數量**：今日 FPC 產量統計
- **環境監測**：溫濕度等環境參數

### 🎮 互動體驗
- **自由視角**：360° 拖拽旋轉、縮放觀看
- **生產模式切換**：自動/手動模式一鍵切換
- **動畫控制**：暫停/播放工廠動畫
- **視角重置**：快速回到最佳觀看角度

## 🚀 技術亮點

### 前端技術
- **Three.js**：高效能 3D 圖形渲染
- **WebGL**：硬體加速圖形運算
- **響應式設計**：適應各種螢幕尺寸
- **即時動畫**：60FPS 流暢動畫效果

### 視覺效果
- **動態照明**：多光源照明系統
- **陰影投射**：真實感陰影效果
- **材質渲染**：金屬、塑料質感模擬
- **粒子效果**：環境霧氣營造工業氛圍

## 🎨 設計理念

這個專案體現了「數位雙生」（Digital Twin）的概念，將真實的 FPC 生產工廠以數位化方式呈現。透過 3D 視覺化，管理者可以：

1. **直觀監控**：一眼掌握整個工廠運作狀況
2. **預測維護**：提前發現潛在設備問題  
3. **效率優化**：分析生產流程瓶頸
4. **培訓教學**：新員工快速熟悉工廠環境

## 📱 使用說明

1. **載入頁面**：等待 3D 場景完全載入
2. **觀看工廠**：拖拽滑鼠旋轉視角，滾輪縮放
3. **監控數據**：左上角面板顯示即時指標
4. **控制操作**：左下角按鈕控制工廠運作

### 控制按鈕
- **🔄 切換生產模式**：自動/手動生產模式
- **📷 重置視角**：回到預設觀看角度  
- **⏯️ 暫停/播放**：控制動畫播放狀態
