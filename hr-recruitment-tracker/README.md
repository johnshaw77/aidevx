# 人才招募追蹤系統 | HR Recruitment Tracker

專為人資部門設計的現代化招募管理平台，提供完整的候選人生命週期管理、面試安排、流程追蹤等功能。

## 🎯 功能模塊

### 📊 控制面板 (Dashboard)
- **關鍵數據統計** - 候選人總數、開放職缺、今日面試、月錄取數
- **實時活動追蹤** - 最新的招募活動和狀態更新
- **一目了然概覽** - 重要指標和待辦事項

### 👥 候選人管理 (Candidates)
- **候選人資料庫** - 完整的候選人信息管理
- **智能搜索** - 按姓名、職位、技能快速搜索
- **多維度篩選** - 按狀態、職位、來源等條件篩選
- **狀態管理** - 待審核、面試中、已錄取、不錄取

### 💼 職缺管理 (Positions)
- **開放職位列表** - 管理所有招募中的職位
- **需求追蹤** - 監控每個職位的招募進度
- **職位詳情** - 職責說明、技能要求、薪資範圍

### 📅 面試安排 (Interviews)
- **面試排程** - 安排面試時間和地點
- **面試官分配** - 指派合適的面試官
- **會議室預約** - 協調會議空間使用
- **狀態追蹤** - 待面試、已確認、已完成

### 🔄 招募流程 (Pipeline)
- **看板式管理** - 直觀的流程階段展示
- **拖拽操作** - 輕鬆移動候選人狀態
- **流程階段**：
  - 待審核 (Resume Review)
  - 面試中 (Interview)
  - 待決策 (Decision Pending)
  - 已錄取 (Hired)

### 📅 行事曆 (Calendar)
- **面試時程** - 月曆式面試安排視圖
- **重要日期** - 關鍵招募節點提醒
- **資源協調** - 避免時間衝突

### 📈 報表分析 (Reports)
- **轉換率分析** - 面試到錄取的成功率
- **招募週期** - 平均招募所需時間
- **成本分析** - 每位候選人的招募成本
- **體驗評分** - 候選人滿意度追蹤
