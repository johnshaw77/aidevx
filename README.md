## 架構

- **主機:** DigitalOcean Droplet (SGP1)
- **Web Server:** Nginx
- **部署:** 純靜態 HTML，push 後自動部署

## 開發

```bash
git clone https://github.com/johnshaw77/aidevx.git
cd aidevx

# 改完推上去
git add -A && git commit -m "update" && git push
```

新增專案只要：
1. 建一個新資料夾（如 `new-project/`）
2. 在 `index.html` 首頁加上連結
3. Push — 自動部署到伺服器
