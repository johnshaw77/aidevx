# 🎵 音樂頻譜視覺化 - Music Spectrum Visualizer

一個帶有**多種視覺模式與實時分析**的互動式音樂頻譜視覺化工具。

## 🎨 核心功能

🎵 **音樂播放與分析**
- 上傳任何音頻格式（MP3、WAV、FLAC 等）
- Web Audio API 實時頻譜分析
- 播放速度控制 (0.5x ~ 2x)

🌈 **4 種視覺模式**
- **條形圖 (Bar)**: 經典頻譜柱狀圖，動態上下跳動
- **波形 (Waveform)**: 連續波形曲線，展現音訊起伏
- **圓形 (Circular)**: 360° 圓形頻譜，呈放射狀展開
- **花瓣 (Flower)**: 花瓣式有機形狀，藝術感十足

🎚️ **自訂控制**
- 主題色彩選擇（顏色選擇器）
- 視覺靈敏度調整（0.1x ~ 5x 倍）
- 播放速度調節
- 實時進度條

📊 **實時資訊面板**
- 當前頻率 (Hz)
- 平均音量 (0-255)
- 峰值頻率與強度
- 播放狀態顯示

## ✨ 視覺特色

- **深紫藍星系背景** + Neon 邊框
- **Canvas 2D 實時渲染**，流暢動畫
- **彩虹色階漸變**：頻率越高顏色越熱
- **完整播放控制**：播放、暫停、停止
- **響應式設計**：桌面 / 平板 / 手機

## 🚀 使用方式

1. 點擊「上傳音樂」選擇音頻檔案
2. 選擇你喜歡的**視覺風格**
3. 調整**主題色彩**和**靈敏度**
4. 點擊 ▶ **播放**開始視覺化
5. 欣賞音樂的色彩變舞！

## 🔧 技術亮點

- **Web Audio API** - AnalyserNode 頻譜分析
- **Canvas 2D** - 4 種不同的視覺化算法
- **MediaElement API** - 音訊播放控制
- **HSL 色彩空間** - 動態色相漸變
- **實時性能優化** - requestAnimationFrame 流暢動畫
