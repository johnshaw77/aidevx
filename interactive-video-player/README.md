# 🎬 互動式視頻播放器 - Interactive Video Player

一個帶有**專業級控制與即時資訊**的現代化視頻播放器。

## 🎮 核心功能

📤 **靈活的媒體加載**
- 拖放上傳視頻檔案
- 點擊選擇本地檔案
- 預設範例視頻快速載入
- 支援所有主流視頻格式 (MP4, WebM, Ogg)

▶️ **播放控制**
- 播放 / 暫停按鈕 (快捷鍵: Space)
- 即時進度條（點擊跳轉）
- 緩衝進度顯示
- 當前時間 / 總時長顯示

🔊 **音量與速度**
- 音量控制滑桿 (0-100%)
- 播放速度調節 (0.25x ~ 2x)
- 鍵盤快捷鍵支援
- 實時數值顯示

📱 **高級功能**
- **畫中畫 (PiP)** - 浮窗播放
- **全螢幕** - 沉浸式體驗
- **即時資訊面板** - 解析度、位元率、緩衝、狀態

⌨️ **鍵盤快捷鍵**
| 按鍵 | 功能 |
|------|------|
| `Space` | 播放 / 暫停 |
| `→` | 快進 5 秒 |
| `←` | 快退 5 秒 |
| `↑` | 增加音量 |
| `↓` | 降低音量 |
| `F` | 全螢幕 |
| `M` | 靜音切換 |

## ✨ 視覺設計

- **深紫藍星系背景** + 靛青邊框
- **漸層進度條**：靛青到螢光綠的流暢過渡
- **圓形播放按鈕** + Glow 發光效果
- **完整響應式設計**：桌面 / 平板 / 手機完美適配
- **Neon 風格控制項**

## 🚀 使用方式

1. **上傳視頻**
   - 拖放視頻到播放器區域
   - 或點擊上傳按鈕選擇檔案
   - 或點擊「範例」快速載入

2. **播放控制**
   - 按 ▶ 按鈕或按 `Space` 播放
   - 點擊進度條跳轉到任意位置
   - 調整音量與播放速度

3. **進階功能**
   - 📱 點擊「畫中畫」浮窗播放
   - ⛶ 點擊「全螢幕」沉浸觀看
   - 查看「資訊面板」瞭解視頻技術規格

## 🔧 技術亮點

- **HTML5 Video API** - 原生視頻元素控制
- **Canvas 進度視覺化** - 流暢的動畫反饋
- **Drag & Drop API** - 拖放上傳體驗
- **Picture-in-Picture API** - 浮窗播放支援
- **Fullscreen API** - 全螢幕沉浸模式
- **鍵盤事件系統** - 完整快捷鍵支援

## 📊 支援格式

| 格式 | 副檔名 | 編碼 |
|------|--------|------|
| MPEG-4 | .mp4 | H.264 video + AAC audio |
| WebM | .webm | VP8/VP9 video + Vorbis audio |
| Ogg | .ogv | Theora video + Vorbis audio |
