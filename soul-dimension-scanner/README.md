# ✨ 靈魂維度掃描器 - Soul Dimension Scanner

一個帶有 **Cyberpunk + 靈性風格** 的互動式心靈測驗工具。

## 🎯 核心功能

📊 **12 個深層心靈問題**
- 關於創意、同理心、勇氣、智慧、平衡、成長等核心維度
- 滑桿式互動評分（0-100）

🔮 **6 維度靈魂圖譜**
- Canvas 實時雷達圖視覺化
- 創意力 • 同理心 • 勇氣 • 智慧 • 平衡 • 成長

💬 **個性化靈魂解讀**
- AI 生成的靈性分析文本
- 根據綜合評分輸出激勵座右銘

## 🎨 視覺特色

✨ **Neon 未來主義風格**
- 深紫藍色星系背景 + 閃爍星星
- 靛青 (#00ffff) / 螢光綠 (#88ff00) / 粉紅 (#ff0080) 霓虹配色
- 完整的 Glow 效果與陰影

🎭 **交互動畫**
- 滑桿拖曳即時更新雷達圖
- Loading 旋轉動畫 + 文字漸入效果
- 按鈕 Hover 縮放與發光

📱 **響應式設計**
- 桌面寬屏雙欄佈局
- 平板/手機單欄堆疊

## 🚀 如何使用

1. 在瀏覽器中打開 `index.html`
2. 依次回答 12 個心靈問題（滑桿評分）
3. 點擊「🚀 掃描我的靈魂」按鈕
4. 查看你的靈魂維度圖譜與個性化解讀
