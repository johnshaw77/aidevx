# 🐱 招財貓運勢占卜 | Lucky Cat Fortune

一個充滿東方神秘色彩的運勢占卜網站，由可愛的招財貓為您預測每日運勢！

## ✨ 功能特色

- 🎯 **每日運勢占卜** - 點擊招財貓獲取專屬運勢
- 🌟 **運勢等級** - 大吉、中吉、小吉、平、小凶五個等級
- 🎲 **幸運數字** - 隨機生成 3-6 個專屬幸運數字
- 💡 **開運建議** - 貼心的開運小貼士
- 📱 **響應式設計** - 支援手機、平板、桌機
- 💾 **本地存儲** - 每日運勢自動保存，避免重複占卜
- 🎊 **分享功能** - 將好運分享給朋友
- 🎭 **精美動畫** - 招財貓互動動畫、眨眼效果
- 🎁 **節日特效** - 新年期間特殊視覺效果

## 🎨 設計亮點

- **招財貓造型** - 純 CSS 繪制的可愛招財貓
- **溫暖配色** - 金黃、紅色主題，充滿吉祥寓意  
- **流暢動畫** - 平滑的過渡效果和互動回饋
- **東方美學** - 融合傳統文化與現代設計

## 🛠 技術棧

- **前端**: HTML5, CSS3, Vanilla JavaScript
- **設計**: 響應式設計、CSS Grid & Flexbox
- **動畫**: CSS Animations & Transitions
- **存儲**: LocalStorage API
- **分享**: Web Share API + Clipboard API

## 📱 使用說明

1. 開啟網頁，看到可愛的招財貓
2. 點擊「點擊招財貓，獲取今日運勢」按鈕
3. 觀看招財貓動畫，等待運勢揭曉
4. 查看運勢等級、詳情、幸運數字和建議
5. 使用分享功能將好運傳遞給朋友

## 🌟 特殊功能

### 每日限制
- 每人每天只能占卜一次，保持神秘感
- 運勢結果會保存到隔夜，隔日可重新占卜

### 互動效果  
- 滑鼠靠近招財貓會有 3D 傾斜效果
- 招財貓會定期眨眼，增加生動感
- 點擊時的縮放和旋轉動畫

### 節日彩蛋
- 新年期間會有飄落金幣特效
- 背景顏色會變成節日主題

## 🚀 本地運行

```bash
# 克隆項目
git clone https://github.com/yourusername/lucky-cat-fortune.git

# 進入項目目錄
cd lucky-cat-fortune

# 使用任意 HTTP 服務器運行
# 例如 Python 3:
python -m http.server 8000

# 或者 Node.js:
npx serve .

# 或者直接用瀏覽器開啟 index.html
```
