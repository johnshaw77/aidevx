# 圖片切割器 | Image Cutter Tool

專業的前端圖片裁剪工具，專為Icon製作和圖片尺寸調整而設計。完全基於瀏覽器端技術，無需上傳到服務器，保護隱私安全。

## 🎯 核心功能

### 📤 圖片上傳
- **拖拽上傳** - 直接將圖片拖拽到上傳區域
- **點擊上傳** - 點擊選擇本地圖片文件
- **格式支援** - JPG、PNG、GIF、WebP 等主流格式
- **即時預覽** - 上傳後立即顯示原始圖片

### 📏 尺寸選擇

#### 預設Icon尺寸
- **16×16** - Favicon 網站小圖標
- **32×32** - 小圖標、工具欄按鈕
- **48×48** - Windows 系統圖標
- **64×64** - 桌面應用圖標
- **128×128** - 應用程序圖標
- **256×256** - 高解析度圖標
- **512×512** - Retina 顯示圖標
- **1024×1024** - App Store 應用圖標

#### 自定義尺寸
- **靈活設定** - 任意寬度和高度（1-2048px）
- **即時更新** - 輸入即時生成預覽
- **批量包含** - 自動加入批量導出清單

### 🎨 圖片處理

#### 雙重裁剪模式
- **自動裁剪** - 智能居中正方形裁剪
- **手動裁剪** - 可拖拽調整的裁剪框
- **8方向拖拽** - 支援8個調整手柄精確控制
- **等比例調整** - 按住 Shift 鍵保持寬高比 🔥
- **實時預覽** - 裁剪區域即時尺寸顯示
- **拖拽移動** - 可自由移動裁剪框位置

#### 高品質處理
- **等比例縮放** - 保持圖片比例不變形
- **高品質渲染** - 使用最佳圖片平滑算法
- **防鋸齒處理** - 確保小尺寸圖片清晰度
- **邊界約束** - 自動限制裁剪區域在圖片範圍內

#### 品質控制
- **品質滑桿** - 10%-100% 可調整輸出品質
- **即時預覽** - 調整後立即看到效果
- **檔案大小優化** - 在品質和檔案大小間平衡

#### 單一下載
- **點擊下載** - 點擊預覽圖片即可下載
- **PNG 格式** - 支援透明背景
- **自動命名** - 按照尺寸自動命名檔案

#### 批量下載
- **ZIP 打包** - 一次下載所有選中尺寸
- **選擇性導出** - 可自由選擇需要的尺寸
- **檔案整理** - 自動按尺寸命名和分類

### 基本操作流程
1. **上傳圖片** - 拖拽或點擊上傳目標圖片
2. **選擇裁剪模式** - 自動居中裁剪 or 手動拖拽裁剪
3. **調整裁剪區域** - （手動模式）拖拽調整裁剪框
4. **選擇尺寸** - 點擊預設尺寸或輸入自定義尺寸
5. **調整品質** - 使用滑桿調整輸出品質（可選）
6. **生成預覽** - 系統自動生成所有尺寸預覽
7. **下載結果** - 單個下載或批量ZIP下載
