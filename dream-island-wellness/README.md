# ✨ 靈夢互動療癒島

> 專屬於你的個人療癒空間 | 點擊、創造、冥想

## 🌈 功能特色

- **互動式島嶼畫布** — Canvas 繪製的動態療癒島，支援即時互動
- **靈感元素放置** — 6 種元素可選：靈燈、噴泉、植物、雲彩、夢境、重置
- **粒子效果系統** — 每次點擊都會綻放療癒的靈光粒子
- **AI 療癒語錄** — 隨機推送 10 段靈性肯定語
- **實時統計** — 追蹤你的靈感觸點數與島上元素數量
- **動態視覺效果** — 元素會隨時間動態浮動、發光、呼吸

## 🎮 怎麼玩？

1. **自由點擊畫布** — 直接點擊島嶼背景，隨機放置靈感碎片
2. **選擇元素放置** — 點擊上方按鈕選定元素，再點擊畫布放置
3. **觀察動畫** — 所有元素都會動起來（火焰搖曳、水流噴射、植物搖晃）
4. **獲得啟發** — 每次互動都會推送新的療癒語錄
5. **重置空間** — 點擊「重置」按鈕清空島嶼，重新開始

## 🎨 設計理念

- **Cyberpunk 禪意風格** — 靈性與現代科技的融合
- **配色系統** — 紫藍漸變背景 + 青綠 (#a8edea) + 粉紅 (#fed6e3) 的療癒色彩
- **流動美學** — 所有元素都在呼吸和浮動，營造寧靜的冥想感

## 📊 核心技術

- **Canvas 2D 繪圖** — 動態繪製島嶼與元素
- **CSS 動畫** — 粒子上升、元素浮動、漸變文字
- **遞迴繪製** — RequestAnimationFrame 維持 60FPS 動畫
- **交互式數據追蹤** — 實時統計靈感觸點
