# 🎨 心情顏色產生器

> 描述你的心情，讓 AI 為你創造專屬的情感色彩

## ✨ 特色功能

- **🎭 智慧情感分析**：輸入心情描述，AI 自動分析並生成對應色彩
- **🌈 漸變色彩生成**：每種心情都有獨特的漸變色彩組合
- **📚 色彩心理學**：提供專業的色彩情感理論解釋
- **💡 快速範例**：內建多種心情範例，一鍵體驗
- **📱 一鍵分享**：分享你的專屬心情色彩到社群媒體

## 🎯 使用方式

1. 在輸入框中描述你當下的心情
2. 點擊「生成我的心情色彩」按鈕
3. 欣賞 AI 為你創造的專屬色彩
4. 閱讀詳細的色彩心理學分析
5. 複製 CSS 代碼或分享給朋友

## 🎨 支援的心情類型

### 積極情緒
- 開心、快樂、愉悅 → 陽光橘黃系
- 興奮、激動 → 活力彩虹系
- 愛、溫暖、感動 → 玫瑰粉紅系

### 平靜情緒
- 平靜、安詳 → 湖水綠藍系
- 舒適、溫馨 → 溫暖米橘系

### 深層情緒
- 憂鬱、悲傷 → 深海藍調系
- 焦慮、緊張 → 焦躁橘紅系
- 複雜心境 → 神祕紫藍系

## 🧠 技術原理

使用先進的自然語言處理技術：
- **關鍵字匹配**：識別情感相關詞彙
- **語意分析**：理解文本的整體情感傾向
- **色彩映射**：基於色彩心理學將情感轉換為視覺表現
- **動態生成**：即時創建獨特的漸變色彩效果
